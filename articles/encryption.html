ENCRYPTION

In this article, how to apply encryption will be explained. What it is, how to use it practically etc.

The Key-Pair Model

Of all available encryption methods, one very common and reliable kind is the private public keypair model. The idea is that a user has two keys; one private that can decrypt (unlock) messages meant only for the owner, and public that encrypts (locks) a message meant for a specific recipient.

Gnu Privacy Guard

Assymetric encryption can be achieved on a linux machine using the gpg terminal utility. Below follows an example of how one might use the gpg command line tool to encrypt an arbitrary input-file to be decryptable only by the intended recipient.

  $ gpg --encrypt input-file --armor --recipient UID --output output-file
