C LANGUAGE

The C Programming Language, designed by Dennis Richie in 1973. It's an imperative language, designed for use close to the hardware level. Still supported and developed to this day. Primarily standardized by as ANSI C.

DOCUMENTATION

  • GNU project complete reference documentation website
  • Advanced Level technical documentation hosted at devdocs.io. useful lookup reference for users with a fair familiarity
  • Beginner Level tutorial at Tutorialspoint.com. Good initial overview of components and syntax. Primarily useful for beginners.
  • The C Programming Language, 2ed, Kernighan & Richie. A useful fundamental reference for the language co-authored by the language's designer.

BASIC FILE LAYOUT

In order for a C file to compile at all, first make it look like this:

#include <header-file.h> #include 'library-file.c' /* declare functions before defining them up here */ int main(int argc, char const *argv[]){ //Main program code goes here return some_int; }

COMPILING

In a linux environment, the goto compiler is gcc, easily invoked with:

gcc path/to/program.c -o path/to/executable
Note that if the input does not have a file ending, the language must be specified with the -x option.

C++ LANGUAGE

According to the author, C++ was always meant to be an expansion to C, and not a separate language. As it is, C++ is a object oriented compiled language that works real fast and introduces a significant amount of very useful native functionality that significantly modernizes the C language. A great deal of overlap exist, where considerable support for C programs, syntax and resources exist directly in the language itself.

DOCUMENTATION

For beginners, the tutorialspoint wiki might be the appropriate starting point. The official reference is hosted on the horrendously ugly main website. A slightly more attractive documentation is avalable at devdocs.io. Microsoft have also compiled a fairly nice language reference.

FILE LAYOUT

#include <cstdlib> #include <iostream> using namespace std; int main(int argc, char const *argv[]){ /*Program content*/ return some_int; }

COMPILING

c++ files can be compiled using gcc. It requires a link specification, but otherwise works essentially the same way a c program would

gcc path/to/program.cpp -lstdc++ -o path/to/executable
The -lstdc++ option is necessary to appropriately link the c++ standard libraries.
